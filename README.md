GDG[x] Hub
===

[![Build Status](https://travis-ci.org/gdg-x/hub.png?branch=master)](https://travis-ci.org/gdg-x/hub)

## Install dependencies
```
npm install
bower install
```

## Start Debug Server (with Livereload)
```
grunt serve
```

## Build the Hub
```
grunt build
```
The distributable version will be placed in /dist
